Project Description

This project is a web-based Inventory Management System developed with Flask and SQLite. The application allows users to manage company assets, employees, and assignments through a clean CRUD interface.

The system supports:

  • Employee management
  • Asset management
  • Asset assignment and return tracking
  • CSV import for employees and assets
  • Duplicate prevention and validation
  • Assignment history tracking
  • Asset status management
  • Sorting and filtering of records

The application uses a layered architecture: 

routes -> services -> models

How to Use

1. Start the Application

python app.py

Then open: 

http://127.0.0.1:5000

2. Asset Management

  • Add new assets
  • Edit existing assets
  • Delete assets
  • Import assets from CSV
  • View assignment history
  • Sort assets by category, brand or status

3. Employee Management

  • Add employees
  • Edit employee data
  • Delete employees
  • Import employees from CSV
  • Sort employees by name or department

4. Assignment Management

  • Assign assets to employees
  • Return assigned assets
  • View active assignments
  • Track assignment history

CSV Import Format

Employees CSV

Assets CSV

Installation Guide

1. Clone Repository

git clone <repository-url>

2. Navigate into Project

cd final-proiect

3. Create Virtual Environment

macOS / Linux

python3 -m venv venv
source venv/bin/activate

Windows

python -m venv venv
venv\Scripts\activate

4. Install Dependencies

pip install flask

5. Run Application

python app.py
